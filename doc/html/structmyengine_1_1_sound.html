void myengine::Sound::onInitialize ( )
virtual

Initialises the Sound.

Initialises the sound by getting the path and adding the required .ogg extension. This gets passed through and loads the sound file.

Attention
Must be a .ogg file, no other format is supported.
Warning
Will throw an exception if no file with the tried extension can be found.
