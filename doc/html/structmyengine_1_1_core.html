◆ start()

void myengine::Core::start ( )

Starts the engine.

Creates the needed utilities and then starts the main loop of the engine. It captures SDL poll events, including: SDL_Quit, SDL_MOUSEMOTION, SDL_KEYDOWN and SDL_KEYUP.

The environment->tick() function gets called to calculate deltatime. Then updates the world and display of the game engine by calling the tick and display function within Entity.
