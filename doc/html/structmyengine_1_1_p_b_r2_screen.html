void myengine::PBR2Screen::onTick ( float  _deltaTime)
virtual

Updates Screen.

Waits for button press to swap screens. When button is pressed, it flags all entities to be destroyed. Then it creates the new screen and sets it up the same way.

Warning
If you spam between the screens, it causes a loop and wil make it become unresponsive.
